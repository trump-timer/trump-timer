# Trump Timer Public Roadmap

This roadmap summarizes product directions we are evaluating or actively improving for [trumptimer.us](https://trumptimer.us). It is not a promise of delivery dates, pricing, model access, or final feature scope.

## Now

- Keep the live presidency timer and January 20, 2029 countdown clear and readable.
- Maintain source labels, dates, and caveats for polling and current-event pages.
- Improve navigation between evergreen trackers and trend-responsive explainers.
- Avoid unsupported claims in political, legal, health, or public-safety contexts.

## Next

- Add clearer update notes for pages that depend on changing public information.
- Improve topical explainers for searchers who need quick context before deeper reading.
- Review structured data for event, article, and reference pages.
- Make source-review requests easier to file through GitHub issues.

## Later

- Evaluate a public source log for high-change pages.
- Improve comparison views for timelines, speeches, and policies.
- Add more durable archive context for pages tied to short news cycles.
- Document editorial standards for countdown, polling, and explainer pages.

## Product Principles

- Dates must be explicit and easy to verify.
- Political and polling claims need source context and caveats.
- The site should remain a reference hub, not a private opinion log.
- This repository is for public product feedback, not private production source code.
