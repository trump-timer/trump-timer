# Trump Timer Public Roadmap

This roadmap describes product areas under active consideration for trumptimer.us. It is not a release commitment.

## Current Focus

- Keep countdown and reference pages clear, readable, and easy to navigate.
- Improve public-source context for trackers and explainers.
- Maintain mobile readability for long public-interest articles.
- Make issue reporting and content correction requests easier.

## Planned Areas

- Better source notes on tracker and explainer pages.
- More consistent cross-linking between countdown, approval, policy, and reference pages.
- Accessibility improvements for keyboard navigation, contrast, and screen-reader labels.
- Clearer separation between reference facts, explainers, and interactive pages.

## Feedback Wanted

Open an issue if a page has outdated context, a broken link, unclear wording, or a missing source that would improve the public reference value.
